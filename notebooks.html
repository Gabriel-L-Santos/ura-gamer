Este é o modelo ideal para jogadores iniciantes ou para aqueles que procuram um notebook gamer acessível e de boa qualidade. Ele vem equipado com uma GPU integrada, processador i5, 8GB de RAM e armazenamento SSD, garantindo assim desempenho satisfatório para jogos leves e aplicativos cotidianos.
